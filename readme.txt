# 📝 Blogging Platform

A simple **full-stack blogging platform** built with **Node.js**, **Express**, and **vanilla JavaScript**.  
It supports user authentication, post creation, scheduling, and comment moderation — all in a clean, responsive interface.

---

## 🚀 Features

✅ User Roles – **Admin**, **Author**, and **Reader**  
✅ Secure **Login / Register** with JWT authentication  
✅ **Create, Edit, Delete** blog posts  
✅ **Schedule posts** for future publishing  
✅ **Search** and filter posts by keyword or tag  
✅ **Comment System** with Admin moderation  
✅ **Likes, Views, and Analytics** tracking  
✅ Simple **RESTful API** endpoints  
✅ Ready to deploy on **Vercel / Render / Netlify**
